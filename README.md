## 前端博客源码

### 指令
#### 安装
```
$ npm install hexo-cli -g
$ npm install
$ git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly # 需要安装hexo-theme-butterfly主题，否则打开是空白页
```
```shell
$ hexo s                 # 本地预览
$ hexo new <title>       # 创建新文章
```

### 发布
提交代码至远程仓库，自动完成部署。
